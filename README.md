## 🏋️ Workout Timer - HIIT Interval Training

Ein moderner, vollständig funktionsfähiger HIIT-Timer mit intuitiver Bedienung, Sound- und Vibrationsfunktionen sowie responsive Design.

![Workout Timer](https://img.shields.io/badge/Version-1.0.0-blue)
![License](https://img.shields.io/badge/License-MIT-green)

### ✨ Features

- ⏱️ **Vollständiger HIIT-Timer** mit konfigurierbaren Intervallen
- 🎯 **Anpassbare Parameter**:
  - Anzahl der Runden (1-99)
  - Arbeitsdauer (5-300 Sekunden)
  - Pausendauer (5-300 Sekunden)
  - Vorbereitungszeit (0-60 Sekunden)

- 🔊 **Sound & Vibration**:
  - Countdown-Beeps in den letzten 3 Sekunden
  - Benachrichtigungsbeeps bei Phasenwechsel
  - Optionale Vibrationsfunktion für mobile Geräte

- 📊 **Visuelle Indikatoren**:
  - Großes animiertes Display mit Countdown-Timer
  - Kreisförmiger Fortschrittsbalken für die aktuelle Phase
  - Gesamtfortschrittsbalken für das gesamte Training
  - Echtzeit-Runden- und Phasenübersicht

- 🎮 **Steuerung**:
  - Start/Pause/Reset-Buttons
  - Tastaturkürzel (Leertaste zum Starten/Pausieren, ESC zum Zurücksetzen)
  - Intuitive +/- Buttons für schnelle Anpassungen

- 📱 **Responsive Design**:
  - Optimiert für Desktop, Tablet und Mobilgeräte
  - Touch-freundliche Bedienelemente
  - Anpassbare Benutzeroberfläche

### 🚀 Schnellstart

1. **Klone das Repository**:
   ```bash
   git clone https://github.com/plukin-00/workout-timer.git
   cd workout-timer
   ```

2. **Öffne die Webseite**:
   - Öffne `index.html` direkt in deinem Webbrowser, oder
   - Verwende einen lokalen Webserver (z.B. `python -m http.server 8000`)

3. **Konfiguriere dein Training**:
   - Stelle die Anzahl der Runden ein
   - Passe die Arbeitsdauer und Pausendauer an
   - Aktiviere/deaktiviere Sound und Vibration nach Bedarf

4. **Starte dein Training**:
   - Klicke auf "▶️ Start" oder drücke die Leertaste
   - Der Timer zählt zunächst die Vorbereitungszeit herunter
   - Danach beginnt das HIIT-Training mit Arbeitsphasen und Pausen

### 🎮 Tastaturkürzel

| Taste | Aktion |
|-------|--------|
| **Leertaste** | Start / Pause |
| **ESC** | Zurücksetzen |

### 📋 Trainingsablauf

1. **Vorbereitung** ⏳ - Countdown vor dem Start (Standard: 10 Sekunden)
2. **Arbeit** 💪 - Intensive Trainingsphase (Standard: 30 Sekunden)
3. **Pause** 😮‍💨 - Erholungsphase (Standard: 15 Sekunden)
4. **Wiederholung** - Arbeit und Pause wiederholen sich für jede Runde
5. **Fertig** 🎉 - Training abgeschlossen

### 🔧 Technologie

- **HTML5** - Semantisches Markup
- **CSS3** - Responsive Design mit Gradients und Animationen
- **Vanilla JavaScript** - Keine Abhängigkeiten, reines JavaScript
- **SVG** - Für den animierten Fortschrittsring
- **Web Audio API** - Für Sound-Benachrichtigungen
- **Vibration API** - Für Vibrationsfeedback auf mobilen Geräten

### 💡 Tipps für optimales Training

- **Anfänger**: Beginne mit 3-5 Runden, 20s Arbeit, 20s Pause
- **Fortgeschrittene**: 8-10 Runden, 40s Arbeit, 20s Pause
- **Profis**: 12+ Runden, 45s+ Arbeit, 15s Pause
- Nutze die Vorbereitungszeit, um dich mental vorzubereiten
